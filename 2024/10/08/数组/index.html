数组

1. 数组概念

1.1 数组

用来存储一组相同类型数据的数据结构，如{1, 2, 3}，{“aa”, “bb”}

数组本身属于引用数据类型

1.2 元素

数组中存储的数据称为元素

元素可以是任何数据类型

1.3 数组长度

数组元素的个数称为数组的长度

数组创建好之后，长度是不能改变的

1.4 下标/索引**

下标又称为索引，是指元素在数组中的位置

下标从0开始

2. 静态创建**

数组使用[]表示，所谓静态创建是指在创建数组的同时对数组元素进行初始化赋值。

元素的数据类型[] 数组变量名 = 数组常量。

int[] array = {1, 2, 3};

[]的位置在变量名前后都可以，如int array[]，但是规范写法在前。

3.1 获得数组长度

3.2 数组元素的访问

通过数组元素的下标获得数组元素

如果下标超过范围，会出现数组下标越界的异常( ArrayIndexOutOfBoundsException)

3.3 循环遍历

普通for循环

增强for循环

4. 动态创建**

数组的内存存储方式

数组声明

声明数组时只是指明数组元素的类型，没有指定数组的大小，也没有给数组分配存储数据的内存空间。

数组元素类型[] 数组名;

String[] names;

数组初始化

数组名 = new 数组元素类型[数组长度];

数组元素的初始值是其元素类型的默认初始值。

names = new String[10];

System.out.println(names[0]);

元素赋值

names[0] = “小张”；

5. 二维数组

5.1 概念

一维数组中的每个元素又是一个一维数组

{1, 2, 3} //一维数组

{ //二维数组

{1, 1},

{2, 2},

{3, 3}

}

5.2 二维数组的声明**

语法格式：

int[][] array;

5.3 二维数组的初始化**

规则数组，规定行数和列数

array = new int[3][2]; //第一个中括号[]声明行数，第二个中括号[]声明每行列数

不规则数组，只规定最外层行数，在分别指定每行的列数

array = new int[3][];

array[0] = new int[2];

array[1] = new int[3];

array[2] = new int[4];

5.4 二维数组的赋值**

通过下标给每个元素单独赋值

int[][] array = new int[3][2];

arryay[0][0] = 2;

5.5 二维数组的访问**

行数

array.length

每行列数

array[0].length

循环访问二维数组元素

6. 数组排序

6.1 冒泡排序

每位数字依次和后面的数字比较，小的互换位置。

优点：简单

int[] array = {1, 0, 3, 5, 2};

for (int i = 0; i < array.length - 1; i++) { //对数组元素遍历

for (int j = i+1; j < array.length; j++) { //每个元素依次和后面的比较，一直比较到最后一个

​ if (array[i] > array[j]) { //如果前面数大

​ int temp = array[i];

​ array[i] = array[j];

​ array[j] = temp;

​ }

}

}

6.2 选择排序

每位数字依次和后面的数字比较，记录小的位置，最后和该位数字互换

优点：比冒泡排序减少了替换次数

int[] array = {1, 0, 3, 5, 2};

for (int i = 0; i < array.length - 1; i++) { //对数组元素遍历

int minIndex = i; //假设当前最小

for (int j = i+1; j < array.length; j++) { //每个元素依次和后面的比较，一直比较到最后一个

​ if (array[minIndex] > array[j]) { //如果前面数大

​ minIndex = j; //重写记录最小位置

​ }

}

if (i != minIndex) {//i和最小位置替换

​ int temp = array[i];

​ array[i] = array[minIndex];

​ array[minIndex] = temp;

}

}

6.3 插入排序

从第二位开始，每位和前面的数字比较，如果大结束，如果小则替换。

优点：对于相对有序数组，替换次数较少

7. 元素查找

7.1 顺序查找

遍历数组，从数组中第一个元素开始查找。最简单，平均效率低

7.2 二分法查找

使用二分查找法的前提是数组必须是有序的，原理是反复和中间值比较。
