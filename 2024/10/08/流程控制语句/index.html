流程控制语句

1. 系统输入

创建扫描器

Scanner sc = new Scanner(System.in); //创建扫描器，扫描系统输入

读取数据
读取数值：nextByte(), nextShort(), nextInt(), nextLong(), nextFloat(), nextDouble()
读取字符串：next()
int num = sc.nextInt(); //如果用户没有输入，程序会一直等待
String str = sc.next();
关闭扫描器
sc.close(); //关闭扫描器，释放资源

2.流程控制的三种结构

顺序结构：默认的，从上到下依次执行

分支结构：满足某个条件会被执行

循环结构：满足某个条件反复执行

3. if

3.1 语法格式

3.2 语法规则

判断if条件是否成立，如果成立则执行该分支，然后结束if语句。如果不成立，则判断以下else if条件是否成立，如果成立则执行，如果不成立则继续向下判断，以此类推…. 以上都不成立，则执行else分支。

案例1：输入年龄判断是否成年案例
案例2：输入成绩，判断成绩等级

3.3 注意事项

4. switch

4.1 语法格式

4.2 执行流程

判断变量/表达式的值是否等于case的常量，如果相等则执行该分支，然后碰到break终止switch语句。如果不相等，则继续向下判断，以此类推。。。以上条件都不成立时，执行default。

案例：输入数字，输入1则打印“开始游戏”，输入2则打印“暂停游戏”，输入3则打印“退出游戏”。其他输入则打印“输入错误”

4.3 注意事项

表达式支持的类型：byte、short、int、 char、String、枚举
case子句中的取值必须是常量，且取值必须是不同的
default子句是可选的
break语句用来在执行完一个case分支后使程序跳出switch语句块
如果不加break，则语句会继续执行，直到找到break
当多个条件采用相同处理方式时，可以共用break

5. while

5.1 语法格式

while(循环条件)｛
循环体;
}

5.2 执行流程

循环条件是一个布尔表达式，先判断循环条件是否成立，如果成立执行循环语句。循环后再次判断循环条件是否成立，以此类推…

案例：一根绳子100米，每次剪断1半，问多少次后长度小于5米（不考虑浮点数）#

6. do-while

6.1 语法格式

do {
循环体;
} while (循环条件);

6.2 执行流程

先执行一次循环体，再判断循环条件是否成立，如果成立再次执行循环语句。然后再次判断循环条件是否成立，以此类推…

7. for

7.1 语法格式

for (初始化①; 循环条件②; 条件变更③)｛
循环语句④;
｝

7.2 执行流程

首先执行初始化①，然后判断循环条件②。如果不成立，循环结束。如果成立执行循环语句④。然后执行条件变更③。然后再次判断循环条件。以此类推…

案例1：打印0到9的数字
案例2：计算0到9的和

7.3 for 和 while

for循环和while循环对于循环逻辑都可以实现，for循环一般用于循环次数已知的情况，while循环一般用于循环次数未知的情况。
