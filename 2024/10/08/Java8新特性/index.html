java23

函数式接口（functional interface）：内部有且只有1个抽象方法

  1. 函数式接口中由于只有1个抽象方法，那么该抽象方法可以表示该接口。

Lambda表达式：又称为闭包，又称为箭头函数。允许将Lambda表达式作为方法的参数使用。
语法：

  1. 基本语法
    (形参) -> {业务代码}
  2. 形参：如果只有1个形参，可以省略小括号
  3. {}业务逻辑：如果只有1行代码，可以省略大括号
  4. {}业务逻辑: 如果只有1行代码，而且return，可以省略大括号和return

forEach循环：java8新增的，对集合实现循环的方式

stream: 流，是数据进行处理的。（数组和集合对数据进行存储的）

stream的创建

  1. Collection： stream()
  2. arr: Arrays.stream(arr)
  3. Stream.empty()
    Stream.of(1, 2, 3)

stream的中间操作: 操作之后获得新的Stream,原有的stream被关闭

steam的终端操作: 操作之后获得一个结果,原有的stream被关闭

接口新特性

java8之前,接口的内部的组成部分

  1. 公有的静态常量 public static final 数据类型 常量名 = 值;
  2. 公有的抽象方法 public abstract 返回类型 方法名(参数);

java8 新增了2种组成成分

  1. 公有的静态方法
  2. 公有的默认方法

Base64: java8之前属于第三方的类库,java8合并到java8的sdk中.

Base64的作用: 编码和解码

  • 编码: 将二进制的数据数据转成标准格式数据
    解码: 将编码后的二进制数据还原回原始数据

    编码:

    • Encoder 编码器,可以编码
    • 编码, encode()

    解码:

    • Decoder 解码器
    • 解码, decode()

    实现数据的加密和解码, 不安全的加密

LocalDateTime

java8之前处理日期和时间

  • Date, SimpleDateFormat, Calendar
  • 存在时区的概念

java8新增 LocalDateTime

  • 可以处理整体日期,可以处理单独的字段
  • 不存在时区概念

LocalDate: 只包含年月日
LocalTime: 只包含小时分钟秒
用法和LocalDateTime完全一致

方法引用： java8中提供了更加简洁的方法调用方法，运算符 ::

方法引用分类

  1. 引用静态方法

  2. 引用对象的实例方法

  3. 引用类的实例方法

  4. 引用构造方法
