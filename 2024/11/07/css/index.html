三、选择器

（一）选择器概述

  • 选择器是CSS规则作用的对象，CSS规则由选择器和属性声明构成。

（二）选择器分类

  1. 基本选择器
    • *：通用元素选择器，匹配任何元素。
    • E：标签选择器，匹配指定标签元素。
    • .val：class选择器，匹配class属性等于val的元素。
    • #val：id选择器，匹配id属性等于val的元素。
  2. 关系选择器
    • E,F：多元素选择器，匹配E或F元素。
    • EF：交叉选择器，条件并且关系。
    • E F：后代元素选择器，匹配E元素后代的F元素。
    • E > F：子元素选择器，匹配E元素的子元素F。
    • E ~ F：同级元素选择器，匹配E元素之后的同级F元素。
    • E + F：毗邻元素选择器，匹配紧随E元素之后的同级元素F。
  3. 属性选择器（CSS3）
    • E[att]：属性名称选择器，匹配具有att属性的E元素。
    • E[att=val]：属性值选择器，匹配att属性等于val的E元素。
    • E[att~=val]：匹配att属性具有多个空格分隔值且其中一个等于val的E元素。
    • E[att|=val]：匹配att属性具有连字号分隔且以val-开头的E元素。
    • E[att^=val]：属性att的值以val开头的元素。
    • E[att$=val]：属性att的值以val结尾的元素。
    • E[att*=val]：属性att的值包含val字符串的元素。
  4. 伪类选择器
    • a:link：正常连接样式。
    • a:hover：鼠标放上去的样式。
    • a:active：鼠标按下时的样式。
    • a:visited：已访问过的样式。
    • E:enabled：匹配表单中激活的元素。
    • E:disabled：匹配表单中禁用的元素。
    • E:checked：匹配表单中被选中的radio或checkbox元素。

（三）选择器特性

  1. 继承性：子元素继承父元素文字相关样式。
  2. 叠加性：不同类型选择器可同时作用于一个标签。
  3. 优先性：!import >行间样式> id > class >标签> 通配，选择越精准优先级越高。
  4. 重要性：使用!important可将选择器优先级设为最高。

四、常用样式

（一）颜色

  • 可用英文名称（如green、red）、RGB（如rgb(255,0,0)）、RGBA（如rgba(255,255,255,1)）、十六进制（如#ff0000、#f00）表示。

（二）字体

  1. font-family：设置字体外观，可指定多个字体，如font-family: serif, sans-serif;。
  2. font-size：设置字体尺寸，如14px或相对于父元素的百分比。
  3. font-style：有normal（正常）、italic（斜体）、oblique（倾斜）。
  4. font-weight：取值范围100 - 900，或normal、bold、bolder、lighter。
  5. font简写方式：如font: italic bold 14px/1.5 serif;。

（三）文本

  1. text-decoration：可设置underline（下划线）、overline（上划线）、line-through（删除线）、none（无修饰）。
  2. text-transform：有capitalize（单词首字母大写）、uppercase（转换为大写）、lowercase（转换为小写）。
  3. text-align：left（左对齐）、center（居中对齐）、right（右对齐）。
  4. text-indent：使用em进行缩进。
  5. line-height：可用长度值或百分比指定行高。
  6. text-shadow（CSS3）：属性值有h-shadow、v-shadow、blur、color。
  7. word-wrap（CSS3）：属性值break-word可对长单词换行。

（四）背景

  1. background-color：设置背景颜色，用法同color。
  2. background-image：通过url(图片路径)设置背景图片。
  3. background-repeat：有repeat-x（横向平铺）、repeat-y（纵向平铺）、repeat（横向和纵向平铺）、no-repeat（不平铺）。
  4. background-position：可使用长度值、百分比或方位词（如left、center、right、top、bottom）指定背景位置。
  5. background-size（CSS3）：可使用长度值、百分比、auto、cover、contain定义背景尺寸。
  6. background简写方式：如background: #f00 url(bg.jpg) no-repeat center;。
  7. background-attachment：scroll（滚动）或fixed（固定）。
  8. CSS精灵/背景精灵：将多个小图融合到大图，通过背景定位设置小图背景。
  9. 多重背景（CSS3）：如background: url(bg_flower.gif) no-repeat left top, url(border.png) no-repeat right bottom;。

（五）列表

  1. list-style-type：设置列表标记样式，如none（无标记）、disc（实心圆）、circle（空心圆）、square（实心方块）、decimal（数字）等。
  2. list-style-position：outside（标记在文本外）或inside（标记在文本内）。
  3. list-style-image：使用url指定列表项标记图像。
  4. list-style简写方式：如list-style: square inside url(marker.gif);。

（六）表格

  • 常用border-collapse: collapse合并表格和单元格边框。

（七）2D与3D转换（CSS3）

  1. 2D转换
    • translate(px,py)：元素位移。
    • rotate(deg)：元素旋转。
    • scale(倍数)：改变元素大小。
    • skew(deg,deg)：元素倾斜。
    • matrix()：组合2D变换。
  2. 3D转换
    • rotateX(deg)、rotateY(deg)、rotateZ(deg)：使元素绕相应轴旋转。

（八）CSS3与浏览器内核

  • 不同浏览器采用不同内核，编写CSS3样式时可能需加前缀，如-moz-（firefox）、-ms-（IE）、-webkit-（safari、chrome）、-o-（Opera）。

五、CSS布局

（一）元素显示

  1. 显示分类
    • 块级元素：独占一行，可设宽高，如p、div等。
    • 行级元素：共享一行，不可设宽高，如a、span等。
    • 行内块级元素：共享一行，可设宽高，如img。
  2. display属性：可改变元素显示类型，如inline（转行元素）、block（转块元素）、inline-block（转行内块级元素）、none（隐藏且不占空间）。
  3. visibility属性：hidden可隐藏元素但占用空间。

（二）盒子模型

  1. 盒子模型相关概念
    • 宽高（width，height）：包含元素内容的宽度和高度。
    • 边框（border）：可设置宽度、颜色、样式。
    • 内边距（padding）：内容到边框的距离。
    • 外边距（margin）：边框与其他盒子的距离。
  2. width/height宽高：对块级和行内块级元素有效，width默认父容器100%，height未设置时根据内容自适应。
  3. border边框
    • border-width：设置边框宽度。
    • border-color：设置边框颜色。
    • border-style：设置边框样式（如solid、dashed等）。
    • border简写方式：统一设置边框。
    • 单独设置某边样式：如border-top: 1px solid red;。
    • border-radius（CSS3）：设置圆角边框。
    • box-shadow（CSS3）：设置边框阴影。
  4. padding内边距：可分别设置四个方向或统一设置，如padding: 10px 20px;（上下10px，左右20px）。
  5. margin外边距：类似padding，可设置四个方向或统一设置，如margin: 0 auto;（水平居中）。
  6. 全局reset取消默认内外边距：使用*{margin:0; padding:0;}。
  7. overflow溢出处理：hidden（超出隐藏）、scroll（显示滚动条）、auto（超出下拉显示）。
  8. outline轮廓：绘制于元素周围，可突出元素，如outline: none;取消输入框焦点默认边框。

  1. 浮动理解：float使元素漂浮脱离文档流，如左浮动示例。
  2. 左浮动和右浮动：元素可向左或向右浮动，浮动后转为行内块级元素。
  3. float的停止：碰到父级元素边界或其他浮动元素停止，可实现多种布局效果。

（六）定位

  1. 定位概述：position属性规定元素定位类型，绝对定位或固定定位时元素生成块级框，相对定位相对于默认位置偏移。
  2. relative相对定位：参照原位置移动，原空间保留，移动时可盖住其他元素。
  3. absolute绝对定位：参照最近有定位属性的父级元素移动，完全脱离文档流，一般需给父级设相对定位并设宽高。
  4. fixed固定定位：以body为定位对象，不随文档流滑动。
  5. z-index堆叠顺序：仅对绝对定位元素有效，数字越大越靠前，默认0，可为负数。

（九）flex弹性布局（CSS3）

  1. 弹性布局概述：确保元素在不同屏幕和设备上有恰当行为的布局方式，由弹性容器和弹性子元素组成。
  2. 弹性盒子：通过display: flex或inline-flex定义，父级宽度处理有默认和自适应规则。
  3. 弹性盒子常用属性
    • flex-direction：指定子元素排列方式（row、row-reverse、column、column-reverse）。
    • flex-wrap：设置子元素超出父容器时是否换行（nowrap、wrap、wrap-reverse）。
    • flex-flow：flex-direction和flex-wrap的简写。
    • align-items：设置子元素垂直对齐方式（stretch、center、flex-start、flex-end、baseline）。
    • justify-content：设置子元素水平对齐方式（flex-start、flex-end、center、space-between、space-around）。
  4. 弹性子元素属性
    • order：设置子元素排列顺序。
    • flex-grow：设置子元素扩展比率。
    • flex-shrink：设置子元素收缩比率。
    • flex-basis：设置子元素伸缩基准值。
    • flex：flex-grow、flex-shrink和flex-basis的简写。
    • align-self：覆盖容器align-items属性。

六、编码规范

  1. 关键位置添加注释。
  2. 合理换行缩进，排版工整。
  3. 命名不含汉字，不以数字开头，建议英文小写，多单词用中划线分割。
  4. CSS选择器优先用class选择器，最外层可用id选择器，尽量用后代选择器组合。
  5. 使用简写命名需遵循普遍认可的缩写习惯，命名要见名知意，如页面结构、导航、功能、电子商贸等方面有常用命名规范。
