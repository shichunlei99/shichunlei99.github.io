JavaScript基础方法整理

JavaScript基础方法整理

  1. 变量声明与作用域相关方法
    • var：在ES6之前用于声明变量，作用域为全局或函数级，可重复声明。
    • let：ES6中用于声明变量，作用域为块级，不能重复声明。
    • const：ES6中用于声明常量，值一旦设定不能更改，作用域为块级。
  2. 数据类型判断方法
    • typeof：用于检测数据类型，返回值为字符串，如typeof 20返回"number"。
    • isNaN()：判断是否为非数字，只判断字面的数值，如isNaN('abc')返回true。
  3. 字符串方法
    • charAt(index)：返回指定位置的字符。
    • concat(stringX,stringX,...,stringX)：连接字符串。
    • indexOf(searchvalue,fromindex)：从前往后检索字符串，返回首次出现的下标，找不到返回 -1。
    • lastIndexOf(searchvalue,fromindex)：从后向前搜索字符串，返回首次出现的下标，找不到返回 -1。
    • match(subst/regexp)：找到一个或多个正则表达式的匹配。
    • replace(regexp/substr,replacement)：替换与正则表达式匹配的子串。
    • search(regexp)：检索与正则表达式相匹配的值，返回下标，找不到返回 -1。
    • split(separator,howmany)：把字符串分割为字符串数组。
    • slice(start,end)：提取字符串的片断，并在新的字符串中返回被提取的部分。
    • substr(start,length)：从起始索引号提取字符串中指定数目的字符。
    • substring(start,end)：提取字符串中两个指定的索引号之间的字符。
    • toLowerCase()：把字符串转换为小写。
    • toUpperCase()：把字符串转换为大写。
    • Number.parseInt(str)：将字符串转换为整数。
    • Number.parseFloat(str)：将字符串转换为浮点数。
    • new String(str).toString()：将Number对象转换为字符串（Number对象本身也有toString方法）。
  4. 数组方法
    • concat(arrayX,arrayX,......,arrayX)：连接两个或更多的数组，并返回结果。
    • join(separator)：把数组的所有元素放入一个字符串，元素通过指定的分隔符进行分隔。
    • reverse()：颠倒数组中元素的顺序。
    • sort()：对数组的元素进行排序。
    • pop()：删除并返回数组的最后一个元素。
    • push(newelement1,....,newelementX)：向数组的末尾添加一个或更多元素，并返回新的长度。
    • shift()：删除并返回数组的第一个元素。
    • unshift(newelement1,....,newelementX)：向数组的开头添加一个或更多元素，并返回新的长度。
    • splice(index,count,item1,.....,itemX)：删除元素，并向数组添加新元素。
    • forEach((item, index, arr) => {})：遍历数组，参数为回调函数，回调函数参数1是数组元素，参数2是元素下标，参数3是循环的数组。
    • map((item, index, arr) => {})：映射数组，一般与return配合，返回新的映射数组。
    • filter((item) => {})：过滤数组，返回符合条件的元素组成的新数组。
    • find((item) => {})：查找数组中符合条件的元素，若有多个符合条件的元素，则返回第一个元素，找不到返回undefined。
    • findIndex((item) => {})：查找数组中符合条件的元素索引，若有多个符合条件的元素，则返回第一个元素索引，没找到返回 -1。
    • indexOf('指定元素')：查询指定元素的下标，找不到返回 -1。
    • some((item) => {})：数组里面某一个元素符合条件，返回true。
    • every((item) => {})：数组里面所有的元素都要符合条件，才返回true。
    • includes('指定元素')：判断数组是否包含指定数据，包含返回true，否则返回false。
    • Array.from('类数组对象')：把类数组对象转成数组。
    • Array.of(1,2,3,4)：把一组值转成数组。
    • fill(填充的东西,开始位置,结束位置（不包含）)：数组填充。
  5. 对象相关方法
    • 对象赋值定义对象，如var stu = {name: '小张', age: 22, eat: function() {console.log('吃饭了...')}, study: function() {console.log('学习了...')}}。
    • 使用对象.属性或对象.方法()调用对象的属性和方法。
    • Object.assign(target, source_1,...)：用于将源对象的所有可枚举属性复制到目标对象中。
  6. 日期方法（Date类）
    • new Date()：创建当前时间的时间对象。
    • new Date(时间戳)：创建指定时间戳的时间对象。
    • getDate()：返回Date对象的一个月中的每一天，其值介于1 - 31之间。
    • getDay()：返回Date对象的星期中的每一天，其值介于0 - 6之间。
    • getHours()：返回Date对象的小时数，其值介于0 - 23之间。
    • getMinutes()：返回Date对象的分钟数，其值介于0 - 59之间。
    • getSeconds()：返回Date对象的秒数，其值介于0 - 59之间。
    • getMonth()：返回Date对象的月份，其值介于0 - 11之间。
    • getFullYear()：返回Date对象的年份，其值为4位数。
    • getTime()：返回自某一时刻（1970年1月1日）以来的毫秒数（时间戳）。
  7. 数学方法（Math类）
    • Math.PI：圆周率。
    • Math.abs(x)：返回x的绝对值。
    • Math.ceil(x)：向上取整。
    • Math.floor(x)：向下取整。
    • Math.round(x)：四舍五入。
    • Math.max(x,y,z,...)：返回最大值。
    • Math.min(x,y,z,...)：返回最小值。
    • Math.random()：返回0（包括）到1（不包括）之间的随机小数。
    • Math.pow(x,y)：返回x的y次幂。
    • Math.sqrt(x)：返回x的平方根。
  8. 函数相关方法（部分与函数特性相关）
    • 函数声明：function函数名([参数1][,参数2]...){这里是要执行的代码[return返回值;]}。
    • 函数调用：函数名()。
    • 箭头函数：(参数) => {函数体}，当参数只有1时，可以省略（），当函数体内只有1条语句时，可以省略{}，函数体内有return时，且只有1行代码，可以省略return和{}。
  9. 其他方法
    • prompt("提示信息")：获得键盘输入，返回值为字符串类型。
    • parseInt(字符串)：将字符串转换为整数。
    • alert(信息)：弹窗输出信息。
    • document.write(内容)：输出到页面，会以html语法解析。
    • console.log(内容)：将文本输出到控制台。

ES6新增方法整理

  1. 函数相关方法
    • 参数默认值：在函数定义时为参数设置默认值，如function show(msg1, msg2="abc"){console.log(msg1, msg2); }。
    • 可变参数：用...变量名表示，只能放在参数组最后且有且只有一个，在函数内部作为数组处理，如function show(num1,...num2){console.log(num2);}。
  2. 解构赋值方法
    • 数组解构：
      • 基本解构：let [a, b, c] = [1, 2, 3];。
      • 嵌套解构：let [a2, [b2, c2]] = [1, [2, 3]];。
      • 配合扩展运算符：let [a4,...b4] = [1, 2, 3];。
      • 配合默认值：let [a5, b5 = 10, c5 = 0] = [1, 2];。
    • 对象解构：
      • 基本解构：let {name, age} = {"name": "abc", age: 12};。
      • 嵌套解构：let {name2, age2, book2} = {name2: "abc", age2: 12, book2: {price: 100}};。
      • 配合扩展运算符：let {name3,...data3} = {name3: "abc", age3: 12, sex3: true};。
      • 配合默认值：let {name4, age4 = 18} = {name4: "abc"};。
  3. 字符串方法
    • includes(参数字符串)：返回布尔值，判断是否找到参数字符串。
    • startsWith(参数字符串)：返回布尔值，判断参数字符串是否在原字符串的头部。
    • endsWith(参数字符串)：返回布尔值，判断参数字符串是否在原字符串的尾部。
    • repeat(重复次数)：返回新的字符串，表示将字符串重复指定次数返回。
    • padStart(指定长度, 补全字符串)：返回新的字符串，表示用参数字符串从头部（左侧）补全原字符串。
    • padEnd(指定长度, 补全字符串)：返回新的字符串，表示用参数字符串从尾部（右侧）补全原字符串。
    • 模板字符串：用反引号【 】，可定义多行字符串，可在字符串中加入变量和表达式，如let msg = ${name}今年${age + 1}岁;`。
  4. 数组方法
    • Array.from(类数组对象)：把类数组对象转成数组。
    • Array.of(一组值)：把一组值转成数组。
    • fill(填充的东西,开始位置,结束位置（不包含）)：数组填充。
  5. 对象方法
    • 对象属性和方法简写：let p2 = {name, age, eat() {console.log("吃饭了..."); }}。
    • 扩展运算符：let newP = {...person, sex: true};，用于取出参数对象所有可遍历属性然后拷贝到当前对象，前后顺序无所谓，后面的属性会覆盖前面的属性。
  6. 类相关方法（class关键字相关）
    • 类的定义：class Person{constructor(name, age) {this.name = name;this.age = age;} eat() {console.log('吃饭了...'); } sleep() {console.log('睡觉了...'); }}。
    • 继承：class Student extends Person{constructor(name, age, gender) {super(name, age);this.gender = gender;} eat() {console.log('学生吃大餐了...'); }}。
  7. Promise相关方法
    • new Promise((resolve, reject) => {})：Promise的参数是一个回调函数，内部有两个形参resolve（表示成功）和reject（表示失败）。
    • then((result) => {})：接受成功回调信息。
    • catch((error) => {})：接受失败信息。
    • finally(() => {})：不论成功失败都会触发。
  8. 异步处理相关方法（async/await）
    • async：用于声明一个函数是异步的，函数会默认返回一个Promise对象。
    • await：用于等待一个Promise对象的结果，只能放在async函数内部使用。
